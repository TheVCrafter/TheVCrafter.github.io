Formula 1 Voice Assistant

This project parses live F1 telemetry and enables voice commands to provide smart strategic advice based on the current race situation. It connects to a local AI and is designed to improve the game experience by making communication between race engineer and driver more realistic. The user can talk to the engineer using speech-to-text (STT) and receives AI-driven advice through text-to-speech (TTS). The program is still in development and is therefore not fully functional yet.

Wintris

This game delivers a Tetris-like experience with a solid design and a variety of customization options. It stores user data in a local database, displays current high scores, and allows personalized key bindings and volume settings through the settings menu. Currently, the game only works on my PC because I didn't use relative paths when implementing it. I plan to improve this in the future and change the database type from MS SQL to SQLite to make it available for others as well.
