Heute habe ich meine Website fertiggestellt. Zunächst habe ich überprüft, ob alle Links und Buttons korrekt funktionieren. Danach widmete ich mich der Weiterentwicklung des Designs. Ich ersetzte den bisher verwendeten Hintergrund durch einen animierten GIF-Hintergrund. Dabei blieb ich bei meiner Grün-Schwarzen Farbpalette, entschied mich jedoch für ein etwas helleres Grün.

Im Anschluss lud ich eine passende Schriftart herunter: die „Share Tech Mono Regular“. Diese steht unter einer Open-Font-Lizenz und war somit frei verfügbar. Außerdem passt die Schriftart hervorragend zu meinem geplanten Cyber-Design.

Für jede Unterseite fügte ich anschließend passenden Inhalt ein. Die gesamte Website schrieb ich auf Englisch, da diese Sprache gerade im Bereich Informatik besonders wichtig ist.

Anschließend kümmerte ich mich erneut um das Styling. Dabei achtete ich besonders darauf, das Design einheitlich zu halten und mich weiterhin an meine Farbpalette zu halten. Mithilfe von AI implementierte ich außerdem eine einfache Suchfunktion in JavaScript.

Nachdem ich die Website auf GitHub hochgeladen hatte, bemerkte ich einen Fehler: Die Website erstreckte sich nicht über die gesamte Bildschirmhöhe, sondern zeigte einen weißen Rand. Dies konnte ich beheben, indem ich den Standard-Margin und Padding auf 0 setzte und die Höhe des Bodys auf `100vh` (100 View Height) festlegte. Zusätzlich fügte ich im Main-Element `flex: 1;` ein, damit sich die Hauptsektion automatisch an die Höhe anpasst.
